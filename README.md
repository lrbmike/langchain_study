# langchain_study
Langchain 学习手册

## 环境配置
项目 .env 文件配置
```python
#openai key
OPENAI_API_KEY=....
#openai请求域名(反代地址)
OPENAI_API_BASE=....
```

## 环境安装

* 安装项目运行依赖组件包

```python
pip install -r requirements.txt
```
手册使用 Jupyter Notebook , 它是一个开源的Web应用程序，详细使用介绍可查阅 [jupyter官方网站](https://jupyter.org/)

* 安装 Jupyter

```python
pip install jupyter
```

## 运行 Jupyter

使用手册前需要先运行 Jupyter

```python
jupyter notebook
```
